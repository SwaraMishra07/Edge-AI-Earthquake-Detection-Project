# Data Management Protocol\n\n## Overview\n\nThis document defines how data is collected, processed, stored, and versioned for the Edge AI Earthquake Detection research project.\n\n## Data Classification\n\n### 1. Raw Data\n- **Location**: `data/raw/`\n- **Format**: CSV, HDF5, or binary\n- **Retention**: Permanent archive\n- **Access**: Read-only (write-protected)\n- **Sources**:\n  - USGS earthquake catalog\n  - Regional seismic networks (IRIS)\n  - Field deployment sensors\n\n### 2. Processed Data\n- **Location**: `data/processed/`\n- **Format**: NumPy arrays (.npy) or HDF5\n- **Retention**: Current project cycle\n- **Generation**: From preprocessing pipeline\n- **Documentation**: Processing parameters recorded\n\n### 3. External Data\n- **Location**: `data/external/`\n- **Purpose**: Third-party datasets, reference data\n- **License**: Properly documented\n- **Citation**: Required in CITATION.cff\n\n### 4. Checkpoints\n- **Location**: `data/checkpoints/`\n- **Purpose**: Model and training state snapshots\n- **Versioning**: Timestamped with experiment ID\n- **Cleanup**: Automated archiving after experiment\n\n## Data Preprocessing Pipeline\n\n### Step 1: Data Ingestion\n\n```python\nfrom app.preprocessing import create_sliding_window, normalize_signal\nimport numpy as np\n\n# Load raw data\nraw_data = np.load('data/raw/seismic_signals.npy')\nprint(f"Raw data shape: {raw_data.shape}")\n```\n\n### Step 2: Windowing\n\n```python\nwindow_size = 200  # Validation Checks\n\n```python\ndef validate_data(data: np.ndarray, expected_shape: tuple):\n    """Validate data quality and shape."""\n    assert data.shape == expected_shape, f"Shape mismatch: {data.shape} vs {expected_shape}"\n    assert not np.any(np.isnan(data)), "Data contains NaN values"\n    assert not np.any(np.isinf(data)), "Data contains infinite values"\n    assert data.dtype in [np.float32, np.float64], f"Unexpected dtype: {data.dtype}"\n    print("✓ Data validation passed")\n```\n\n### 2. Statistical Validation\n\n```python\ndef check_statistics(data: np.ndarray):\n    """Verify data statistics."""\n    stats = {\n        'mean': np.mean(data),\n        'std': np.std(data),\n        'min': np.min(data),\n        'max': np.max(data),\n        'q25': np.percentile(data, 25),\n        'median': np.median(data),\n        'q75': np.percentile(data, 75),\n    }\n    print("Data Statistics:")\n    for key, value in stats.items():\n        print(f"  {key}: {value:.6f}")\n    return stats\n```\n\n## Data Privacy and Ethics\n\n### 1. Sensitive Data\n- Earthquake data is public information\n- No personal information included\n- All sources properly credited\n\n### 2. Data Retention Policy\n- Raw data: Permanent\n- Processed data: Until experiment complete + 1 year archive\n- Temporary files: Automatic cleanup after 30 days\n- Backups: Weekly automated backup to secure location\n\n### 3. Access Control\n- Read access: All team members\n- Write access: Data owner + project lead\n- Delete access: Project lead only\n\n## Data Sharing\n\n### Publishing Datasets\n\n1. **Zenodo Archive**\n   ```bash\n   # Create metadata\n   zenodo-upload --metadata data/MANIFEST.json data/processed/\n   ```\n\n2. **Citation Format**\n   ```bibtex\n   @dataset{earthquakedetection2024,\n     title={Edge AI Earthquake Detection Dataset},\n     author={Your Team},\n     year={2024},\n     doi={10.5281/zenodo.xxxxx}\n   }\n   ```\n\n## Data Lineage Tracking\n\n### Processing History\n\n```json\n{\n  "file": "data/processed/features.npy",\n  "created": "2024-01-15T12:30:00Z",\n  "source_file": "data/raw/seismic_signals.npy",\n  "processing_steps": [\n    {\n      "step": 1,\n      "function": "create_sliding_window",\n      "parameters": {"window_size": 200}\n    },\n    {\n      "step": 2,\n      "function": "normalize_signal",\n      "parameters": {}\n    },\n    {\n      "step": 3,\n      "function": "extract_features",\n      "parameters": {}\n    }\n  ],\n  "output_shape": [10000, 16],\n  "output_dtype": "float32"\n}\n```\n\n## Troubleshooting\n\n### Missing Data\n1. Check `data/raw/` for source file\n2. Verify preprocessing pipeline ran successfully\n3. Check `.research_artifacts/` for experiment logs\n\n### Data Corruption\n1. Verify file hash: `sha256sum data/processed/features.npy`\n2. Restore from backup: `aws s3 sync s3://backup/data/ data/`\n3. Reprocess from raw data\n\n### Storage Issues\n1. Check disk space: `df -h`\n2. Compress old data: `tar -czf data/archive_2023.tar.gz data/processed/`\n3. Move to external storage if needed\n
