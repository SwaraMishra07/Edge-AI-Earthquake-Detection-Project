# Reproducibility Guide\n\n## Overview\n\nThis guide ensures all research findings can be reproduced and verified across different environments and time periods.\n\n## Environment Setup\n\n### Python Version\n- **Recommended**: Python 3.10 or 3.11\n- **Minimum**: Python 3.9\n- **Maximum**: Python 3.12\n\n### Dependency Pinning\n\nAll dependencies are pinned to specific versions in `pyproject.toml`:\n\n```bash\n# Install exact dependencies\npoetry install\n\n# Or with pip\npip install -r requirements-pinned.txt\n```\n\n## Random Seed Management\n\nFor reproducibility, all random operations use a fixed seed:\n\n```python\nfrom app.config import research_config\nimport numpy as np\nimport tensorflow as tf\nimport random\n\n# Initialize random seeds\nrandom.seed(research_config.random_seed)  # 42 by default\nnp.random.seed(research_config.random_seed)\ntf.random.set_seed(research_config.random_seed)\n```\n\n## Experiment Tracking\n\n### Logging Experiments\n\n```python\nfrom app.utils.experiment_tracking import create_tracker\n\n# Create tracker for this experiment\ntracker = create_tracker()\n\n# Log hyperparameters\ntracker.log_hyperparameters({\n    'learning_rate': 0.001,\n    'batch_size': 32,\n    'epochs': 100,\n})\n\n# Log metrics during training\ntracker.log_metrics({\n    'accuracy': 0.95,\n    'precision': 0.94,\n    'recall': 0.96,\n}, stage='validation')\n\n# Save checkpoints\ntracker.save_checkpoint({'model_weights': ...}, name='epoch_50')\n\n# Finalize experiment\ntracker.finalize({\n    'final_accuracy': 0.967,\n    'training_time_hours': 2.5,\n})\n```\n\n### Artifact Structure\n\n```\n.research_artifacts/\n├── exp_20240101_120000/\n│   ├── hyperparameters.json\n│   ├── training_metrics.json\n│   ├── validation_metrics.json\n│   ├── test_metrics.json\n│   ├── checkpoints/\n│   │   ├── checkpoint_*.json\n│   └── results.json\n```\n\n## Configuration Management\n\n### Environment Variables\n\nCreate `.env` file from template:\n\n```bash\ncp .env.example .env\n```\n\n### Configuration Hierarchy\n\n1. **Defaults**: Hardcoded in `app/config.py`\n2. **Environment**: Loaded from `.env` file\n3. **Runtime**: Passed as arguments to functions\n\n### Key Configurations for Reproducibility\n\n```bash\n# Core settings\nRANDOM_SEED=42\nEXPERIMENT_ID=exp_20240101_120000\nLOG_LEVEL=DEBUG\n\n# Model settings\nMODEL_VERSION=1.0.0\nQUANTIZATION_AWARE=true\nINFERENCE_THRESHOLD=0.7\n\n# Sensor settings\nSENSOR_SAMPLE_RATE=100\nWINDOW_SECONDS=2\n\n# MQTT settings\nMQTT_BROKER=localhost\nMQTT_PORT=1883\n```\n\n## Dataset Management\n\n### Data Versioning\n\nAll datasets should be versioned with checksums:\n\n```python\nimport hashlib\n\ndef get_file_hash(filepath):\n    """Compute SHA256 hash of file."""\n    sha256_hash = hashlib.sha256()\n    with open(filepath, 'rb') as f:\n        for byte_block in iter(lambda: f.read(4096), b''):\n            sha256_hash.update(byte_block)\n    return sha256_hash.hexdigest()\n```\n\n### Data Split Reproducibility\n\nUse stratified train-test-val splits with fixed random seed:\n\n```python\nfrom sklearn.model_selection import train_test_split\n\nX_train, X_temp = train_test_split(\n    X, test_size=0.3,\n    random_state=research_config.random_seed\n)\nX_val, X_test = train_test_split(\n    X_temp, test_size=0.5,\n    random_state=research_config.random_seed\n)\n```\n\n## Model Versioning\n\n### Semantic Versioning\n\nModels follow semantic versioning: `MAJOR.MINOR.PATCH`\n\n- **MAJOR**: Architecture change\n- **MINOR**: Training data update or hyperparameter change\n- **PATCH**: Bug fix or optimization\n\n### Model Metadata\n\nEach model includes `model_metadata.json`:\n\n```json\n{\n  "model_name": "earthquake_detector_v1",\n  "version": "1.0.0",\n  "model_type": "tflite",\n  "input_shape": [1, 512, 3],\n  "output_shape": [1, 2],\n  "quantization": true,\n  "performance": {\n    "accuracy": 0.967,\n    "precision": 0.954,\n    "recall": 0.941\n  },\n  "training_data": {\n    "dataset": "USGS_2024",\n    "samples": 10000,\n    "hash": "sha256_hash_value"\n  }\n}\n```\n\n## Testing for Reproducibility\n\n### Unit Tests\n\n```bash\n# Run all tests\npytest tests/ -v\n\n# Run specific test\npytest tests/test_preprocessing.py::test_normalize_signal -v\n\n# Run with coverage\npytest tests/ --cov=app --cov-report=html\n```\n\n### Integration Tests\n\n```bash\n# Run full pipeline\npytest tests/ -m integration -v\n```\n\n## Verification Checklist\n\n- [ ] Python version matches specification\n- [ ] All dependencies installed from pinned versions\n- [ ] Random seeds set consistently\n- [ ] Experiment ID matches documentation\n- [ ] Dataset checksums verified\n- [ ] Model version matches metadata\n- [ ] Results within expected ranges (±1% tolerance)\n- [ ] Artifacts saved in `.research_artifacts/`\n- [ ] All tests passing\n\n## Troubleshooting\n\n### Results Don't Match\n\n1. Check Python version: `python --version`\n2. Verify random seeds: Check `.env` file\n3. Verify dataset: Check data hash\n4. Clear cache: `find . -type d -name __pycache__ -exec rm -rf {} +`\n5. Reinstall dependencies: `poetry install --no-cache`\n\n### Performance Differences\n\n- Different CPU/GPU may cause minor differences (<0.1%)\n- Use CPU-only mode for consistency: `CUDA_VISIBLE_DEVICES=""`\n- Check TensorFlow version compatibility\n\n## References\n\n- [The Turing Way - Reproducibility](https://the-turing-way.netlify.app/reproducible-research/overview.html)\n- [Google: Reproducible Research](https://research.google/pubs/)\n- [Nature: Code is Science](https://www.nature.com/articles/d41586-021-03812-w)\n
